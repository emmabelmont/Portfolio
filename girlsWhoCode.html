Core 4

The Core4 I learned are variables, loops, conditionals, and functions.

Variable: A variable is used to store a value. There are different types of variables depending on the data type being stored such as integers, decimals, or characters.

Conditional: A conditional is a statement that controls the execution of a code that depends on a condition.

Loop: A loop repeat sequences of code for a fixed number of times or while a condition is true.

Function: A function is a block of code that can be reused to preform a specific action.
